## 🚀 Getting Started

1️⃣ **Clone the repo**
```bash
git clone https://github.com/AdamAbdallah1/cedars-tech-landing
cd cedars-tech-landing

2️⃣ Install dependencies

npm install

3️⃣ Run the dev server

npm run dev

4️⃣ Build for production

npm run build

5️⃣ Deploy to GitHub Pages

npm run deploy
